# Distribution Strategy Guide

Choose the best way to share your calculator based on your audience.

---

## Option 1: Standalone .EXE (Best for General Audience)

### When to Use
- Sharing with non-technical League players
- Posting on general gaming forums/subreddits
- Social media distribution
- Want maximum reach with minimum friction

### Pros
✅ No Python installation required
✅ Single file download
✅ Just double-click to run
✅ Most user-friendly option

### Cons
⚠️ Large file size (~50-70 MB)
⚠️ Windows only
⚠️ Some antivirus may flag (false positive)
⚠️ Requires building on your end

## Option 2: Python Scripts (Best for Technical Audience)

### When to Use
- Posting on developer communities
- Sharing with Python users
- GitHub repositories
- Open source projects

### Pros
✅ Small file size (~200 KB)
✅ Cross-platform (Windows/Mac/Linux)
✅ Easy to inspect/modify code
✅ No antivirus concerns

### Cons
⚠️ Requires Python 3.8+ installed
⚠️ Users must run pip install
⚠️ More technical setup

### How to Share
1. Zip all .py files + requirements.txt + README.md
2. Upload to:
   - GitHub (recommended for code)
   - Google Drive
3. Provide clear installation instructions

## Platform-Specific Recommendations

### Reddit (r/leagueoflegends)
**Use:** .EXE + Scripts
- Most users want easy .exe
- Tech users appreciate source code
- Include VirusTotal scan link

### GitHub
**Use:** Scripts only
- Developers prefer source
- Can add releases with .exe later
- Clone-friendly

### Discord Servers
**Use:** .EXE primarily
- Gaming community wants simple
- Pin message with both options
- Provide quick setup help

### League Forums (Official)
**Use:** .EXE + Scripts
- Mixed audience
- Official forum users cautious of .exe
- Having both builds trust

### YouTube Video
**Use:** .EXE
- Video viewers want quick results
- Pin comment with scripts alternative
- Demo the .exe in video
