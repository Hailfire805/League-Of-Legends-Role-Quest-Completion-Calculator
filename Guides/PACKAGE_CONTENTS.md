## Quick Start Guide

**For Users (When sharing):**
1. Extract all files to a folder
2. Double-click `launch.bat` (Windows) or `launch.sh` (Mac/Linux)
3. Select your lane from the menu

**For Developers:**
All code is open source and thoroughly commented. Each calculator file contains:
- Quest mechanics constants (top of file)
- Calculation functions
- GUI implementation
- Graph generation

Feel free to modify, fork, or improve!
